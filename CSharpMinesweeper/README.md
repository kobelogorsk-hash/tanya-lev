# Сапер (Minesweeper) на C# для Visual Studio

## 📁 Файлы проекта:
- `Minesweeper.csproj` — файл проекта .NET 6 Windows Forms
- `Program.cs` — исходный код игры

## 🚀 Как запустить в Visual Studio:

### Вариант 1: Visual Studio 2022
1. Откройте Visual Studio 2022
2. Выберите **"Открыть проект или решение"** (Open a project or solution)
3. Найдите и откройте файл `Minesweeper.csproj` из папки `/workspace/CSharpMinesweeper/`
4. Нажмите **F5** или кнопку **"Запустить"** (Start) для запуска игры

### Вариант 2: Через командную строку
```bash
cd /workspace/CSharpMinesweeper
dotnet run
```

## ⚠️ Требования:
- **.NET 6 SDK** (или новее) с поддержкой Windows Forms
- **Windows** (так как используется Windows Forms)
- **Visual Studio 2022** (рекомендуется) или VS Code с расширением C#

## 🎮 Управление:
- **Левая кнопка мыши** — открыть клетку
- **Правая кнопка мыши** — поставить/убрать флаг (🚩)
- **R** — перезапустить игру после окончания

## ⚙️ Параметры игры:
- Поле: 10×10 клеток
- Мины: 15 штук
- Цветные цифры (1–8) показывают количество мин рядом
- Автоматическое открытие пустых областей

## 🔧 Если возникает ошибка при запуске:
Убедитесь, что у вас установлен .NET 6 SDK:
```bash
dotnet --version
```

Если не установлен, скачайте с официального сайта: https://dotnet.microsoft.com/download
