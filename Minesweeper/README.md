# Сапер (Minesweeper) на Python

Это классическая игра "Сапер", созданная с использованием библиотеки Pygame.

## Требования

- Python 3.x
- Библиотека pygame

## Установка зависимостей

```bash
pip install pygame
```

## Запуск игры

```bash
python minesweeper.py
```

## Управление

- **Левая кнопка мыши** - открыть клетку
- **Правая кнопка мыши** - поставить/убрать флаг
- **R** - перезапустить игру после окончания

## Правила игры

1. Поле 10x10 с 15 минами
2. Открывайте клетки, чтобы найти безопасные зоны
3. Цифры показывают количество мин в соседних клетках
4. Помечайте мины флагами (правая кнопка мыши)
5. Найдите все безопасные клетки, чтобы победить
6. Не наступайте на мины!

## Особенности

- Автоматическое открытие пустых областей
- Цветные цифры для разных значений
- Индикатор победы/поражения
- Быстрый рестарт клавишей R
